## Getting Started
Install from [nexus](https://jira.odigeo.com/wiki/display/DS/Python+packages+repositories):


```sh
pip install ds-sql-formatter
```

Define vars:

* GOOGLE_APPLICATION_CREDENTIALS: gcloud service account path.
* PROJECT: gcloud project

Run 
```sh
prettysql.py
```
